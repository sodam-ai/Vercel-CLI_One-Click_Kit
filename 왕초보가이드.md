# 왕초보 가이드 — Vercel 원클릭 키트

컴퓨터·명령어가 처음이어도 괜찮아요. **그림처럼 순서대로** 따라 하면 됩니다.
핵심은 딱 하나 — **`시작하기.bat` 더블클릭**.

> 더 자세한 설명/명령어 표/라이선스는 **[README.md](README.md)** 에 있어요.
> 🇬🇧 English step-by-step: **[GUIDE.en.md](GUIDE.en.md)**

---

## 0단계. 이 키트가 하는 일 (1줄)

내가 만든 웹사이트를 **인터넷에 올려주는(배포)** Vercel을, 검은 명령어 창 대신
**한국어 메뉴**로 쓰게 해 줍니다. **관리자 허용(UAC) 창은 뜨지 않아요.**

**미리 있어야 할 것:** Windows 10/11 · 인터넷 연결. (Node.js는 없으면 자동으로 깔아줘요.)

---

## 1단계. 시작하기

1. 이 폴더에서 **`시작하기.bat`** 을 **더블클릭**.
2. 파란 경고(SmartScreen)가 뜨면 → **추가 정보 → 실행**.
   (Windows가 새 파일을 조심하라고 띄우는 **정상 경고**예요.)
3. 한국어 안내 창이 열립니다.

> 화살표 **↑ ↓** 로 고르고 **Enter**, 또는 **번호키**를 눌러도 됩니다.
> 맨 위 **">> 지금 할 일"** 한 줄만 보면 다음에 누를 번호를 알려줘요.

---

## 2단계. 설치하기 (처음 1번만)

1. 메뉴에서 **[1] 설치하기** 선택.
2. **Node.js(필수 부품)** 가 없다면:
   - **Y(자동 설치)** 를 누르면 **관리자 창 없이** 휴대용 Node.js를 알아서 받아 깝니다. (추천)
   - 자동이 안 되면 [nodejs.org](https://nodejs.org/) 가 열려요 → **초록색 LTS 버튼** 으로 받아 설치.
   - 설치 후 **창을 닫고 `시작하기.bat`을 다시** 더블클릭하세요. (새 창이라야 인식돼요.)
3. Node.js가 있으면 검은 설치 창이 잠깐 뜨고 **자동으로 끝납니다.** (3~10분, 창 닫지 마세요.)
   - **이미 설치돼 있으면** 묻지 않고 **바로 끝납니다.** (업데이트는 따로 — 4·6단계 참고)

> 💡 **관리자 허용(UAC) 팝업이 뜨지 않습니다.** 키트도, 자동 Node 설치(휴대용)도
> 모두 내 사용자 폴더에만 설치되기 때문이에요.

---

## 3단계. 로그인 (처음 1번만)

1. 메뉴 **[2] 사용하기 → [1] 로그인**.
2. 인터넷 브라우저가 열립니다. 거기서 **로그인 / 권한 허용**.
3. 끝나면 자동으로 이 창으로 돌아옵니다. 화면 아래에 **"잘 됐어요 + 다음 할 일"** 이 떠요.

> 계정이 없으면 브라우저에서 바로 무료로 만들 수 있어요. (GitHub/이메일)
> 로그인이 "확인 안 됨"으로 보이면 → **[6] 내 상태 점검 → [4] 로그인 정확히 확인**.

---

## 4단계. 인터넷에 올리기 (배포)

1. **내 웹사이트 파일이 있는 폴더**가 필요해요. (예: `index.html`이 든 폴더)
2. 메뉴 **[2] 사용하기 → [4] 인터넷에 올리기(배포)**.
3. **"이 폴더가 맞나요?"** 확인이 떠요 → 맞으면 `Y`, 다른 폴더면 `N` 후 경로 붙여넣기.
4. 둘 중 하나를 고릅니다:
   - 🟢 **미리보기 배포 (추천)** — 임시 주소로 올려 나만 확인. **진짜 사이트는 그대로.**
   - 🔴 **진짜 공개 배포** — 방문자가 바로 보게 됨. **`YES`(대소문자 무관)** 입력해야만 진행.
5. 잠시 뒤 **주소(URL)** 가 나옵니다. 그 주소가 내 사이트예요!

처음 배포면 영어로 몇 가지를 물어봐요. **거의 다 Enter(기본값)** 면 됩니다:

| 영어 질문 | 뜻 | 이렇게 |
|---|---|---|
| Set up and deploy? | 올릴까요? | **Y** + Enter |
| Which scope? | 어느 계정? | 그냥 **Enter** |
| Link to existing project? | 기존 것과 연결? | 처음이면 **N** + Enter |
| Project name? | 이름? | 그냥 **Enter** |
| In which directory? | 어느 폴더? | 그냥 **Enter** |

> 끝나면 화면 아래에 **한국어로 "잘 됐어요 + 다음에 뭘 하세요"** 가 같이 떠요.
> 🔧 도메인·환경변수·로그·되돌리기 같은 **고급 기능**은 **[2] 사용하기 → [5] 고급 기능** 에서 한국어로 바로.

---

## 5단계. 업데이트 (가끔 한 번이면 충분)

Vercel을 최신으로 바꾸고 싶을 때 (관리자 창 없이):
- **RUN.bat** 에서 `11` 업데이트  (또는 `[10] 도구 → 2`)
- 또는 **시작하기.bat** 에서 `[2] 사용하기 → [5] 고급 기능 → U`

> 평소엔 안 해도 됩니다. 오류가 자꾸 나거나 새 기능이 필요할 때만.

---

## 6단계. 다 쓰고 지우고 싶을 때 (제거)

1. 메뉴 **[3] 제거하기** (또는 `UNINSTALL.bat`).
2. 정말 지울지 한 번 더 물어봐요 → **`YES`(대소문자 무관)** 입력.
3. **내가 만든 코드·파일은 안 지웁니다.** Vercel 도구만 깨끗이 제거.

---

## 🆘 막혔을 때 (자주 나오는 문제)

> 💡 **가장 빠른 길:** 시작 화면에서 **[6] 내 상태 점검 / 문제 해결**.
> 지금 내 상태(설치·로그인·인터넷)와 해결법을 한 화면에서 보여주고, "지금 할 일"을 콕 집어줘요.

| 이런 일이 | 이렇게 하세요 |
|---|---|
| "Node.js가 없습니다" | [1] 설치하기 → **자동(Y)**, 또는 nodejs.org에서 LTS 받기 |
| 파란 SmartScreen 경고 | **추가 정보 → 실행** |
| 설치가 자꾸 실패 | **백신 잠깐 끄고**(아래) 다시 / 와이파이 확인 |
| `vercel`을 못 찾음 | 창 닫고 **`시작하기.bat` 다시 실행** (새 창 필요) |
| 영어 오류가 떴어요 | 그 **아래에 한국어로 원인 + 다음 할 일**이 같이 떠요 |
| 로그인이 "확인 안 됨" | [6] → **[4] 로그인 정확히 확인** |
| 화살표가 안 먹혀요 | 그냥 **번호키 + Enter** |
| 영어 창이 무서워요 | 진행 중이란 뜻 — **닫지만 마세요** |

**💉 백신이 설치를 막을 때:**
시작 → **Windows 보안** → **바이러스 및 위협 방지** → **설정 관리** → **실시간 보호 잠깐 끔**
→ `[1] 설치하기` 다시 → 끝나면 **반드시 다시 켜기.**

## 🗂️ 파일이 어디 있나요?

- **시작은 항상** `시작하기.bat` (이 폴더).
- 설명서: `README.md`(한)·`README.en.md`(영)·`왕초보가이드.md`(한)·`GUIDE.en.md`(영) + 각 PDF.
- Vercel은 `%APPDATA%\npm`, 자동 Node는 `%LOCALAPPDATA%\Programs\node-portable` 에 깔려요.

## 한 장 요약

```
시작하기.bat 더블클릭
   → [1] 설치하기  (처음 1번)
   → [2] 사용하기 → 로그인  (처음 1번)
   → [2] 사용하기 → 인터넷에 올리기 → 미리보기(안전)
   → (확인되면) 진짜 공개 = YES 입력
   막히면 → [6] 내 상태 점검 / 문제 해결
```

막히면 화면 맨 위 **">> 지금 할 일"** 한 줄만 보세요. 항상 다음에 누를 번호를 알려줍니다.
